**一个父亲的"为爱发电"之旅**

## 两天，从零到上线

### Day 1：搭骨架

第一天，我把主要精力放在了核心功能上。

首先是单词库。我花了大半天时间，整理了近2万个单词：

- **汇总词汇**：6,465个，涵盖小学、初中、高中
- **RAZ分级词汇**：13,473个，从AA到Z2共29个级别

为什么要有这么多？因为我希望这个App能陪伴孩子从小学用到高中。

然后是学习模式。我设计了两种：

1. **选择题模式**：看中文选英文，适合初学阶段
2. **拼写模式**：看中文写英文，适合巩固阶段

每种模式下，单词都会自动播放发音。孩子不用点任何按钮，就能听到正确的读音。

### Day 2：注入灵魂

第二天，我开始思考一个问题：

**怎样才能让孩子主动想学，而不是被迫去学？**

答案很明确：游戏化。

我设计了三个核心机制：

#### 1. 打卡系统 + 连续奖励

每天学习后可以打卡，连续打卡天数越多，奖励越丰厚。这就像很多健身App一样，通过"断签焦虑"来驱动用户。

但我没有把它做成单纯的"签到"。我的设计是：

- 每天必须学完设定数量的单词才能打卡
- 连续打卡有额外经验值奖励
- 中间断签，天数清零

这样既保证了学习的"量"，又培养了学习的"习惯"。

#### 2. 徽章收集系统

我设计了四种稀有度的徽章：普通、稀有、史诗、传说。

解锁条件各不相同：

- 连续打卡7天，获得"坚持一周"徽章
- 学习100个单词，获得"百词斩"徽章
- 正确率达到90%，获得"神射手"徽章
- 完成50个闯关关卡，获得"闯关达人"徽章

这些徽章就像游戏里的成就系统，让学习有了"收集"的乐趣。

#### 3. 闯关模式

这是我最满意的功能。

我设计了5个难度等级：简单、中等、困难、专家、传说。每个关卡有时间限制、错误次数限制，通关后可以获得星级评价（最高三星）。

更有意思的是，关卡是无限的。通关后，系统会自动生成新的关卡，永远有新的挑战。

## 科学记忆：艾宾浩斯的力量

游戏化只是"表"，科学记忆才是"里"。

我在App中实现了艾宾浩斯遗忘曲线算法：

| 掌握度 | 复习间隔 |
|--------|----------|
| 0级 | 1天后复习 |
| 1级 | 2天后复习 |
| 2级 | 4天后复习 |
| 3级 | 7天后复习 |
| 4级 | 15天后复习 |
| 5级 | 30天后复习 |

每次答题正确，掌握度+1；答错则-1。系统会自动计算每个单词的复习时间，在首页显示"待复习"数量。

这样，孩子不用自己安排复习计划，App会告诉他"今天该复习哪些单词"。

## 小细节，大体验

作为一个父亲，我很在意用户体验的细节。

### 音效反馈

- 答对时，播放"Excellent!"语音，给予即时正反馈
- 答错时，播放下降音调，温和提示错误
- 每个单词都可以点击喇叭图标重新听发音
- 例句也有朗读功能

### 自动打卡

完成一组学习后，系统会自动打卡。孩子不用记得去点"打卡"按钮，学习结束就是打卡完成。

### 每日目标可调

在设置里，家长可以调整每天的单词数量（默认20个）。如果孩子今天时间紧，可以调少一点；如果想多学，也可以调多。
